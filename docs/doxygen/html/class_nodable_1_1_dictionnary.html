Detailed Description

The role of this class is to store make easy to convert a string to a TokenType and a TokenType to a string. We can also use reglar expressions.

ex:

"true" => TokenType::Boolean "boolean" => TokenType::BooleanType "^(true|false)" => TokenType::Boolean
